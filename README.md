# Single Page Application based on Vue 2.0 and electron

## Build Setup


# install dependencies
``` bash
yarn install
```
# serve with hot reload at localhost:8080
``` bash
yarn dev
```
# launch electron with dev mode
``` bash
yarn start
```
# package
``` bash
electron-packager .  CPSWallet   --overwrite --out=./outapp/ --no-prune

```
